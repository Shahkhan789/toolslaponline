🔒 MD5

128-bit hash function. Fast but not secure for cryptographic purposes. Good for checksums and data integrity verification.

🛡️ SHA-1

160-bit hash function. More secure than MD5 but still vulnerable. Being phased out in favor of SHA-2 family.

⚡ SHA-256

256-bit hash function. Part of SHA-2 family. Widely used and considered secure. Used in Bitcoin and many security applications.

🚀 SHA-512

512-bit hash function. Strongest in SHA-2 family. Provides maximum security but requires more computational resources.
