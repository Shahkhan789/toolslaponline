📚 Number System Information

🔢 Binary (Base 2)

Uses only 0 and 1. Fundamental to computer systems and digital circuits. Each digit represents a power of 2.

🔟 Decimal (Base 10)

The standard number system we use daily. Uses digits 0-9, with each position representing a power of 10.

🔠 Hexadecimal (Base 16)

Uses digits 0-9 and letters A-F. Common in programming for representing colors, memory addresses, and byte values.

8️⃣ Octal (Base 8)

Uses digits 0-7. Popular in Unix file permissions and older computer systems. Each digit represents 3 binary digits.
